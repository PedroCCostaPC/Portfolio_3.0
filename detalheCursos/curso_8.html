Seja bem-vindo ao curso completo de TypeScript, onde você vai aprender todos os recursos do superset e também a integrar nos frameworks mais utilizados do mercado.

Alguns dos temas abordados são:
Tipos de dados básicos e complexos;
Narrowing;
Criação de tipos;
Orientação a objetos com TS;
Módulos com TypeScript;
Decorators;
Generics;
E muito mais!

Além de você ter no curso uma exploração das funcionalidades de TS, vamos utilizá-lo com ferramentas em alta no mercado de desenvolvimento de software, ou seja, que as empresas estão utilizando em seus projetos.

Faremos a integração de React com TypeScript e também Express com TypeScript.

Para concretizar o aprendizado tanto nestes frameworks como também no TS, o curso inclui projetos completos.

Com destaque para o projeto final, onde criaremos uma API RESTful, utilizando a arquitetura MVC, banco de dados MongoDB com Mongoose, sistema de logs com Winston e Morgan, validação de dados com Express Validator, e teste da API com Postman.

E claro, tudo isso seguindo os padrões de desenvolvimento impostos pelo TypeScript e boas práticas do mercado de desenvolvimento de software.

Somado a todo este aprendizado, você ainda conta com: exercícios e desafios, material didático de apoio e questionários.

Por último e não menos importante, você terá direito a dois grandes bônus:
Mini curso de JavaScript moderno (ES6+);
Introdução a APIs com Express;

Estes dois tópicos vão te ajudar muito a tirar todo o proveito do curso de TypeScript, e você ganha tudo isso ao adquirir este curso :)
